MDX component that gives instructions on how to add Docs.

Basic Component Usage:

```js
<AddingDocs />
```

For styling and syntax highlighting this can be wrapped in the MDXLayout Component and Theme Provider.

Themed Component Usage:

```js
import { ThemeContext } from '@teambit/documenter.theme.theme-context';
import { MDXLayout } from '@teambit/ui.mdx-layout';

<ThemeContext>
  <MDXLayout>
    <AddingDocs />
  </MDXLayout>
</ThemeContext>;
```
