## Adding Documentation

> Document your components by creating a file with _\*.docs.mdx_ extension.

### DisplayName, Description and Labels

Modify the component name by adding a displayName. Add a description for your component and labels which are used for searching on bit.dev.

```mdx
---
displayName: My Component
description: Description of my component
labels: ['react', 'typescript', 'ui']
---
```

### Code Block and Live Playground

Add a codeblock to show how your component can be used.

```js
import { Hello } from './hello';

() => {
  return <Hello />;
};
```

Add `live=true` to create a Live Playground:

```jsx live=true
() => {
  return <p>Hello!</p>;
};
```
